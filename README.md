YouTube Video Summarizer & AI Query Assistant
This extension lets you summarize and query youtube video.
Just enter your free gemini api key in the extension settings and start using the extension.
You can get the free gemini api key from https://aistudio.google.com/apikey
Features:
•	Summarize YouTube Videos: Quickly generate concise summaries of YouTube videos using the transcript powered by the Gemini API.
•	Chatbot Interface: Ask specific questions about the video content and receive accurate answers instantly.
•	Segment Selection: Select a specific portion of the video using start and end timestamps for tailored summaries and queries.
Whether you're catching up on lengthy lectures or analyzing insightful talks, this extension streamlines your video consumption, saving you time and effort.
